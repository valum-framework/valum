Valum web micro-framework written in Vala

Documentation

Valum provides user documentation to get you started with the framework.

The framework is built upon libsoup and Libgee and automated with the waf build system, so reading about all these is a good start.

The valadoc.org website provides api specification of most libraries available in Vala.
